A minimalistic offline Python Console App for PDF utilities. Includes
- Reduce PDF
- Extract Pages
- Merge PDFs in Box

How to run?
- Clone the repository
- `pip install -r requirements.txt`
- Then double-click on run.bat

Please note that,
- In Merge PDF functionality, you need to name the files such that thay appear in your desired order. Like numbering them sequentially 1,2,3... (something like '1-abc.pdf','2-xyz.pdf',etc...) will do.
